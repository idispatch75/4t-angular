# Installation

Install Node.js and then:

```sh
$ git clone https://github.com/idispatch75/4t-angular.git
$ cd 4t-angular
$ sudo npm -g install grunt-cli karma bower bower-installer
$ npm install
$ bower-installer
$ grunt watch
```

```sh
$ cd server
$ node server.js
```
open http://localhost:1337/ in Chrome with `--disable-web-security`
